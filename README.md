# Snake Game

This is a simple Snake game implemented in Python using Pygame.

## How to Run

1. Make sure you have Python 3 installed.
2. Install Pygame: `pip install pygame`
3. Run the game:
   ```
   python main.py
   ```

## Features
- Classic snake gameplay
- Modular code structure for easy extension
